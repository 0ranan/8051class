## 阶段目标 1 拓展与动手

1. 熟练掌握 电烙铁、恒温台 熟练使用常见的焊接技巧：走锡、飞线、贴片
2. 能够编写简单的 51 程序,如控制 LED 闪烁,按键控制 LED 亮灭等
3. 能够使用立创 EDA 进行电路设计和仿真，如 LED 八位显示
4. 拓展目标：能够使用建模软件对电路板进行简单的外壳设计

## 阶段目标 2 更多单片机编程技巧

### 阶段目标

能够进行较为复杂的 51 程序编写,熟练掌握基本的单片机编程如技巧：

1. 中断
2. 定时器
3. 串口通信
4. io 口的多种模式
5. 原理：视觉暂留
6. 原理：矩阵键盘
7. 原理：按键防抖
8. 能够看简单的电路图

## 阶段目标 3 尝试体验更多的开发环境

1. esp8266 + arduino 开发环境 ，尝试进行物联网项目编程
2. 树莓派开发环境尝试搭建 EMQX 服务器搭建，尝试使用树莓派驱动舵机

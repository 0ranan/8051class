## 文档特点

1. **结构清晰**：每个元件文档都按照统一的结构组织，包括概述、工作原理、类型、参数、应用等
2. **图文并茂**：通过表格和图片（在img文件夹中）直观展示元件特性
3. **实用性强**：包含大量实际应用示例和计算公式
4. **易于理解**：采用通俗易懂的语言，适合电子初学者

## 使用指南

1. 选择感兴趣的电子元件目录
2. 阅读对应的README.md文件了解详细内容
3. 参考图片目录中的示例图片加深理解
4. 结合实际电路设计进行学习和应用
